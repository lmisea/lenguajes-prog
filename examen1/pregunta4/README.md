# Pregunta 4 - Módulo Vec3 (vectores 3D)

Módulo para vectores tridimensionales con operadores sobrecargados.

Archivos principales:
- Módulo: [`vec3.lua`](vec3.lua) (exporta `Vec3`, constructor: [`Vec3.new`](vec3.lua))
- Cliente/demo: [`pregunta4.lua`](pregunta4.lua)
- Tests: [`test_pregunta4.lua`](test_pregunta4.lua)

Uso:
- Ejecutar demo:
  ```
  lua pregunta4.lua demo
  ```
- Ejecutar tests:
  ```
  lua test_pregunta4.lua
  ```
  o
  ```
  lua pregunta4.lua test
  ```
